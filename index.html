Current Project: Fliegengitter Deluxe

This Project started first as an Idea of an Application just to have some forms to automate a data dump into spreadsheets.
It started as an favour for my work colleague, but grew incrementally into a more complex system.

It went from Simple Data to .xlsx-Converter to an web application, that persists custom Orders and recurring Elements in a PostgreSQL Database.

From a single Spreadsheet to 2 different spreadsheets for the Recipients Groups - Supplier and Customers.
Our newest Addition was the Creation of PDF-Invoices, where we take the newest state in our Database for the chosen Customer and calculate the current price, the paid taxes and the net price.
Currently there are updates on the way, Handling Security and additional necessary Functionalities.
